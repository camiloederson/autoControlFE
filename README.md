# AutoControl Frontend

## 📌 Overview
This is the frontend for AutoControl — a responsive Angular application for car shop management.  
It provides an interface for managing clients, mechanics, spare parts, and service scheduling.

## 🛠 Tech Stack
- Angular
- Tailwind CSS
- SweetAlert2
- Local deployment (development phase)

## ✨ Features
- User interface for CRUD operations
- Service scheduling views
- Responsive design with Tailwind CSS
- SweetAlert2 notifications

## 🚀 Setup
```bash
git clone <repo-url>
cd autocontrol-frontend
npm install
ng serve
